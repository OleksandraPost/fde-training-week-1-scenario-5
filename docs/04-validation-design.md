# Validation Design

## Functional Validation
- task lifecycle correctness
- state transitions
- escalation logic

## Safety Validation
- no clinical decision paths
- mandatory escalation for sensitive inputs

## Integration Validation
- system behavior on missing data
- no fallback guessing

## Audit Validation
- all actions logged
- traceable decisions

## Outcome Validation
- reduction in intake defects
- reduction in manual workload

## Failure Modes
- missed escalation
- incorrect task completion
- silent failures
- over-automation
